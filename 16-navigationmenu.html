Navigation Menu

서비스 메뉴 탐색

‘Navigation Menu’ 태스크는 서비스가 제공하는 메뉴를 탐색하는 기능이다. 대분류 메뉴의 수 및 위계에 따라 노출 방식이 상이하다.

'Navigation Menu' 태스크 화면 흐름

'Navigation Menu' 태스크 화면 설계

탭 바 (Tab bar)

대분류 메뉴가 5개 이하인 경우 서비스 하단에 플로팅되는 탭 바를 사용하여 대분류 탐색이 직관적일 수 있도록 지원한다. 사용자의 콘텐츠 입력 등 행위를 유도하는 서비스는 탭 바 내에 콘텐츠 입력(Add) 버튼을 노출하기도 한다. 대분류 메뉴가 5개를 초과하는 경우 선택 영역이 좁아 권장되지 않으나, 탭 바 중 첫 번째 메뉴를 메뉴 페이지를 호출하는 버튼으로 사용(Tab bar with page menu)하고, 나머지 탭 바 메뉴를 메뉴 페이지 내에 노출되지 않는 알림, 설정 기능 등으로 사용할 수 있다.

사이드 드로워 메뉴 (Side drawer)

대분류 메뉴가 5개를 초과하는 경우 일반적으로 사이드 드로워 메뉴로 노출된다. 사이드 드로워 메뉴를 호출하는 버튼은 페이지 상단 플로팅 탭 혹은 하단 탭 바 중 한 곳에 위치한다. 메뉴가 한 페이지를 초과하거나 중분류 이하로 분류되는 경우 전체 페이지로 노출되며, 한 페이지 내에 노출될 수 있는 경우 화면의 일부를 가리는 레이어 형태로 노출된다. 터치 인터렉션으로도 호출할 수 있는 경우 사이드 드로워 메뉴가 노출되는 트랜지션이 사용 방법을 인지시킬 수 있는 가이드로 작용하기도 한다.

탭 (Tab)

중분류 메뉴는 일반적으로 페이지 내 상단 탭으로 표시된다. 보통 중분류 메뉴가 6개를 초과하는 경우 좌우 스크롤을 통해 탐색하도록 설계할 수 있으나 휴먼 에러(human error) 발생 소지가 있어 탭 방식 대신 사이드 드로워 메뉴에서 노출될 수 있도록 설계한다.
